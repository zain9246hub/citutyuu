# Deal Pricing & Features Guide

## Pricing Structure

### Non-Metro Cities

| Tier | Price | Tax |
|------|-------|-----|
| Standard Deal | ₹235 | All taxes included |
| Highlight Deal | ₹589 | All taxes included |
| City-Wide Push | ₹1,770 | All taxes included |

### Metro Cities

| Tier | Price | Tax |
|------|-------|-----|
| Standard Deal | ₹471 | All taxes included |
| Highlight Deal | ₹1,179 | All taxes included |
| City-Wide Push | ₹4,130 | All taxes included |

---

## Tier Features & Capabilities

### 1. Standard Deal (₹235 / ₹471)

**Visibility:**
- Shown in **Explore Deals** page only
- Not featured on home page

**Zip Code Targeting:**
- Minimum: 1 zip code
- Maximum: 2 zip codes

**Notifications:**
- Users within the selected zip codes receive notifications
- Targeted local reach

**Best For:**
- Small local businesses
- Neighborhood-focused promotions
- Budget-friendly advertising

---

### 2. Highlight Deal (₹589 / ₹1,179)

**Visibility:**
- **Featured on Home Page** with "Featured" badge/icon
- Also visible in Explore Deals page
- Premium placement

**Zip Code Targeting:**
- Minimum: 4 zip codes
- Maximum: 6 zip codes

**Notifications:**
- Users within all selected zip codes receive notifications
- Wider local reach

**Best For:**
- Growing businesses
- Multi-area coverage
- Higher visibility campaigns

---

### 3. City-Wide Push (₹1,770 / ₹4,130)

**Visibility:**
- **Featured on Home Page** with premium placement
- Visible in Explore Deals page
- Maximum exposure

**Zip Code Targeting:**
- No zip code selection needed
- Entire city coverage

**Notifications:**
- **All users in the selected city** receive push notifications
- Maximum reach within the city

**Best For:**
- Large businesses
- City-wide promotions
- Grand openings & major sales
- Maximum brand awareness

---
